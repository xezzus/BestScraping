# BestScraping

## Install
```git clone https://github.com/xezzus/BestScraping.git```

## Start

```docker-compose up -d```
## Stop
```docker-compose down```

## Getting started

Send HTTP RAW POST request on http://127.0.0.1:9001
```json
{"url":"http://example.com"}
```
Get response of html document
